# Dissertation Writing Style Preferences

- Tone: formal, concise, direct, academically rigorous.
- Narrative flow: write as a connected story that links sections and prior points.
- Audience assumption: self-contained writing for non-specialist CS markers.
- English variant: British English spelling and grammar.
- Vocabulary: avoid extravagant, ornate, or unusual words.
- Sentence style: prefer medium-to-long sentences with clear logical flow.
- Punctuation preference: comma-friendly style; minimal semicolons and colons.
- Hyphen preference: do not use hyphens as sentence separators; only use hyphens for proper compound terms.
- Preferred connectors: Thus, Furthermore, Moreover, Although, On the one hand, On the other hand, Despite, Following this.
- Explanation style: provide full descriptions with concise examples, then continue the core argument.
- Section clarity: prioritise coherence over verbosity; avoid overlong reporting where brevity improves clarity.
- Voice pattern from prior high-mark coursework: systematic method -> implementation details -> quantified results -> limitations -> conclusion.
- Evidence style: include precise numeric results where available, then interpret them cautiously.
- Critical reflection style: acknowledge limitations directly without weakening valid contributions.
- Structure discipline: keep context conceptual, implementation technical, evaluation analytical.

# Writing Quality Rules for Future Drafting

- Start sections with a clear purpose sentence.
- Keep one main claim per paragraph.
- Link each paragraph to previous context with a connector where useful.
- Avoid repetitive phrasing and avoid inflated language.
- Define key terms before using them in analysis.
- Keep reporting concise; avoid unnecessary speculative discussion in results sections.
- Use consistent tense: past tense for completed work, present tense for established facts.
- Ensure every claim in evaluation is supported by measured evidence.
- Maintain objective, professional tone while preserving narrative flow.
